# WebdriverIO_v5_TypeScript

This is a boilerplate project that uses version 5 of WebdriverIO and TypeScript.

## Getting Started
```
git clone git@github.com:jpolley/WebdriverIO_v5_TypeScript.git
cd WebdriverIO_v5_TypeScript
npm install
npm test
```
